# 🏪 Sistema de Gerenciamento de Mercado

Sistema desenvolvido em **Python** com foco na aplicação dos conceitos de **Programação Orientada a Objetos (POO)**. O projeto simula o gerenciamento de um mercado, permitindo administrar produtos, estoque, clientes e funcionários por meio de um menu interativo em terminal.

## 📋 Funcionalidades

- Cadastro de produtos
- Controle de estoque
- Cadastro de clientes
- Cadastro de funcionários
- Gerenciamento das informações do mercado
- Navegação por menu interativo no terminal

## 🛠️ Tecnologias Utilizadas

- Python 3
- Programação Orientada a Objetos (POO)

## 📂 Estrutura do Projeto

```
gerenciamento-mercado-main/
│
├── main.py           # Inicialização do sistema
├── menu.py           # Menu principal
├── mercado.py        # Classe principal do mercado
├── estoque.py        # Controle de estoque
├── produto.py        # Classe Produto
├── pessoa.py         # Classe base Pessoa
├── cliente.py        # Classe Cliente
└── funcionario.py    # Classe Funcionário
```

## 📚 Estrutura das Classes

O projeto foi organizado utilizando conceitos de orientação a objetos:

- **Pessoa**: classe base para representar pessoas.
- **Cliente**: especialização da classe Pessoa.
- **Funcionário**: especialização da classe Pessoa.
- **Produto**: representa os produtos comercializados.
- **Estoque**: responsável pelo gerenciamento dos produtos.
- **Mercado**: concentra as operações do sistema.
- **Menu**: responsável pela interação com o usuário.

## 🚀 Como Executar

### 1. Clone o repositório

```bash
git clone <url-do-repositorio>
```

### 2. Acesse a pasta

```bash
cd gerenciamento-mercado-main
```

### 3. Execute o programa

```bash
python main.py
```

> Certifique-se de possuir o Python 3 instalado em sua máquina.
